# 143B-File-System
Project 1 for CS 143B class.

## How to Run
If on Linux machine:
```{bash}
cmake --preset linux
```

If on Windows machine:
```{bash}
cmake --preset windows
```

To build the binary, enter:
```{bash}
cmake --build build --target main
```

To execute the binary, enter:
```{bash}
./build/main
```
